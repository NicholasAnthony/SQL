Not Another SQL!

This project demonstrates basic SQL operations using a SQLite database.

Database Structure

The database (demo.db) contains two tables:

  • students: Stores 1,000 student records (id, first_name, last_name, age, grade).
  • courses: Stores 1,000 course enrollments (course_id, course_name, student_id), with each student assigned one course.

Sample SQL Queries

Click here to see the jupyter notebook with queries

Below are examples of SQL queries demonstrated in queries.ipynb:


-- Select all students
SELECT * FROM students;

-- Select students with grade 'High Distinction'
SELECT name, age FROM students WHERE grade = 'High Distinction';

-- List students and their assigned course
SELECT s.first_name, s.last_name, s.grade, c.course_name
FROM students s
JOIN courses c ON s.id = c.student_id;

-- Count students per course
SELECT course_name, COUNT(student_id) AS student_count
FROM courses
GROUP BY course_name
ORDER BY student_count DESC;
